# ProyectoFinal1
KixHub
Cual es el proposito de KixHub? 
El propósito central de KixHub es ser la plataforma comunitaria de compra y venta más confiable y curada para los entusiastas del streetwear, los coleccionistas de zapatillas, y los seguidores de la cultura hypebeast, facilitando transacciones seguras de productos exclusivos y de reventa.

Las tecnologias que utilice en KixHub son: HTML, CSS, y JavaScript

Para entrar y ejecutar el sitio:
Descomprimir el ZIP file
Entrar a Visual Studio
Seleccionar el folder
oprimir "Start Debuging"

Informacion relevante:
El website tiene 7 paginas, 5 de elllas interactivas, 2 de ellas que no lo son por que proveen informacion de la tienda.

El valor principal que KixHub aportará a sus usuarios se centra en la confianza y la experiencia:
•	Confianza Garantizada: Proceso de verificación de autenticidad riguroso para cada artículo, eliminando el riesgo de falsificaciones.
•	Comunidad y Drops Exclusivos: Ofrecer acceso prioritario a lanzamientos limitados o colaborar con marcas/influencers para tener drops exclusivos que no se encuentran en otro lugar.
•	Insights del Mercado: Herramientas que muestren el valor de reventa histórico o actual de las zapatillas más populares, ayudando al usuario a tomar decisiones de compra o inversión.

Publico Objetivo:
El publico obejtivo de KixHub es de 16-35 años (Jóvenes adultos y adolescentes conocedores de las tendencias, fuertemente influenciados por redes sociales y la cultura urbana.)
